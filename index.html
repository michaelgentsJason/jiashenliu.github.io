成果展示 / Research

WebPentestRAG

面向自动化 Web 渗透测试的 RAG 框架。设计 Web Task Graph 管理状态，并结合 grounded generation + dry-run validation， 在任务成功率上相较基线模型取得约 42% 的提升。

University of Warwick · Master Dissertation

MMSearch-R1

基于 GRPO 的多模态检索路由框架，让模型按需搜索并优化中间推理质量；在保持准确率前提下，将不必要检索率从 100% 降至 67.1%。

2025 - Present

MIKRAG System

多意图、多模态 PDF 检索系统。通过意图分解、模态自适应加权与向量量化，在检索精度和效率上实现协同提升。

ColQwen2.5 / BGE / Vespa

DiaKG-GPT

基于数据增强的糖尿病智能问答系统，构建 DiaMediQA 数据集并结合 LoRA SFT 与 RLAIF 训练，显著提升医疗问答质量。
