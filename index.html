关于

我是 Jiashen Liu / 刘家燊，现就读于 University of Warwick（Cyber Security Engineering, MSc with Distinction）。 本科毕业于南京农业大学信息管理与信息系统专业，毕业论文 DIAKG-GPT 获校级优秀毕业论文奖。

我的研究聚焦于多模态 AI与知识密集型推理系统，重点关注视觉-语言-结构化知识融合、工具增强推理与复杂任务多步决策。 近期研究涵盖 Web 安全自动化、文档检索理解与多模态评测基准构建。

成果展示

WebPentestRAG

面向自动化 Web 渗透测试的检索增强框架，集成任务图状态管理与 grounded generation，显著提升复杂攻防任务成功率。

MMSearch-R1

基于 GRPO 的多模态智能检索路由方法，在保持准确率下显著降低不必要检索，提升推理效率与可解释性。

DiaKG-GPT

面向糖尿病问答场景的医疗智能系统，结合数据增强、SFT 与 RLAIF 策略，优化专业医疗问答质量。
